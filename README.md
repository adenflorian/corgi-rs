# Running From Source

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)
- `trunk`
  - Only required to run the web version
  - `cargo install trunk`
- `wasm32-unknown-unknown` target
  - Only required to run the web version
  - `rustup target add wasm32-unknown-unknown`

## Running

`git clone...`

### Desktop

`cargo run`

### Web

`trunk serve`
